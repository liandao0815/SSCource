# SSCourse
## javaweb--学生选课系统
> **ssh框架实现学生选课系统**
## 项目介绍
### 核心功能
* 学生模块
1. 个人信息管理：学生录入个人信息、学生修改资料
2. 选课管理：浏览可选课程、选择课程、查询已选课程
* 管理员模块
1. 课程管理：新增课程、删除课程、修改课程、查询课程
2. 账号管理：新增学生账号、删除学生账号
3. 信息统计：课程选课人数统计、开设课程类别统计
### 开发环境
**操作系统**：Windows 10

**IDE工具**：Eclipse

**JDK**：JDK1.8

**中间件**：Tomcat

**数据库**：MySQL5.7

**后端框架**：Struts2、Spring4、Hibernate5

**前端框架或库**：Jquery、Bootstrap
